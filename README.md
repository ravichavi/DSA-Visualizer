# ⚡ DSA Visualizer (Stack & Linked List)

An **interactive Single Page Application (SPA)** built using **HTML, CSS, and JavaScript** to visualize core Data Structures concepts.

This project focuses on making DSA intuitive through **real-time animations, clean UI, and step-by-step operations**.

---

## 🚀 Features

### 📌 Stack Visualizer

* Push / Pop operations
* TOP pointer visualization
* Animated updates
* Real-time explanation of operations

### 🔗 Linked List Visualizer

* Insert at position
* Delete from position
* Automatic position handling
* HEAD → nodes → NULL representation
* Step-by-step traversal animation with moving pointer

### 🎨 UI/UX Highlights

* Single Page Application (no reload)
* Sidebar navigation with active state
* Smooth transitions & animations
* Ripple button effects
* Clean and minimal design

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, animations)
* JavaScript (DOM manipulation)

## 🧠 Concepts Covered

* Stack (LIFO)
* Linked List (Dynamic structure)
* Traversal algorithms
* DOM manipulation & UI rendering
